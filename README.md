# 📚 Sistema de Banco de Dados para Biblioteca com MySQL Views

Este projeto foi desenvolvido para simular o gerenciamento robusto de uma biblioteca, focando na aplicação prática de **MySQL Views** organizadas por pilares estratégicos: operacional, gerencial, estatístico e segurança.

## 🛠️ Tecnologias Utilizadas
* **Banco de Dados:** MySQL
* **Linguagem:** SQL

## 🎯 Estrutura das Views
O projeto conta com consultas customizadas para diferentes níveis de acesso no sistema:
* **Operacionais:** Consultas diárias de empréstimos e devoluções.
* **Gerenciais:** Relatórios consolidados para tomada de decisão.
* **Estatísticas:** Análise de dados e comportamento dos usuários (ex: categorias menos lidas).
* **Segurança:** Ocultação de dados sensíveis de usuários para consumo da aplicação.

## 🚀 Como Executar o Projeto
1. Clone este repositório.
2. Importe o arquivo `script_biblioteca.sql` no seu gerenciador MySQL (Workbench, DBeaver, etc.).
3. Execute as consultas para testar o comportamento das Views.
